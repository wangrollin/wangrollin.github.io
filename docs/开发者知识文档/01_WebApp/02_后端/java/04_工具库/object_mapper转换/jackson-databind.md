## 代码

### 【bean的代码】

#### 忽略某些字段

```java
@JsonIgnore
```


#### 让某些字段变成key的时候，首字母大写

```java
@JsonProperty(value = "Type")
```


#### 忽略某些值为null的字段

```java
@JsonInclude(JsonInclude.Include.NON_EMPTY)
```
